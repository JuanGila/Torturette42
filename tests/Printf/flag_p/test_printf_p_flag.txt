1.- Puntero A Variable int
    Descripcion: Se comprueba que se muestra un puntero a varable de tipo int.
    OK: Se muestra correctamente un puntero a varable de tipo int.
    KO: No coincide la salida con printf o el valor de retorno es incorrecto.
    CRASH:
        Sigsev en caso de ¿?
        Sirargv en caso de ¿?
    /*
    void *ptr = 0x1234abcd;
          ↓
    imprime: "0x1234abcd"
    */
    int x = 42;
    printf("%p\n", &x);
    y comparar el resultado con -> printf("%lx\n", (unsigned long)&x);(Eso te dará una pista bastante clara de cómo suele implementarse %p internamente.)
